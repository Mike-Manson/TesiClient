Serious Game Taxonomy

A tree view of the classification of serious games proposed by this experimental project. The root is the serious game which is divided into six categories which are sets of objects.

This taxonomy extends the G/P/S Model to include part of the four-dimensional model of the International Journal of Computer Games Technology.
the G/P/S model takes into consideration the gameplay, the purpose and the scope of serious games, but does not consider the type of platform with which it is possible to interact with the game, nor the various age groups of the players at to which they are directed, which are included in the four-dimensional model of the journal.

In addition, BoardGameGeek was inspired by the categorization of the games in the various reference areas and, given that this site categorizes only physical games, the last dimension was designed which includes the platform with which you can interact with serious games.
